Home, dark. The brass nib indicator is the one warm point of color — your eye goes straight to which pen is live.

"Inkwell" is a placeholder name — rename freely (trademark reasons rule out Neo / LAMY in the store listing). Every color, type role, and the dot-grid motif are specified as tokens in the companion design-system file so this can be implemented exactly in Compose. The bold lives in one place — ink on the Ncode grid — and everything else stays quiet on purpose.
